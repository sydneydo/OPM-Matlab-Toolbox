5. OPM Notation Operations 

    5.6 Delete Relation/Link

For fundamental relations (characterizations, inheritances, aggregations, and instantiations), click the line which connects the relation symbol to the destination thing and then click the delete button of the keyboard. If the same type of relation connects a specific source to several destinations, then only the selected line will be removed. Otherwise the relation symbol along with the lines to the source and the destination will be removed.

For the other types of relations and links, simply select the relation/link and press the delete button of the keyboard.
