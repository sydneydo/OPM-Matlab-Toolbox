5. OPM Notation Operations 

    5.2 Control States

After confirming the object addition, the State tab of the object can be activated. Pressing the Add button in this tab opens the state properties window, where you should enter the state name and optional description (the General tab). In the Preferences tab you can specify whether the state is initial and/or final and its time constraints (the min and max duration time of the state). In the Misc tab you can change the state border, fill, and text colors.

From the States tab of an object, you can also remove a state or change its properties by selecting it and clicking the Remove or Props buttin, respectively. You can also choose to leave the states arrangement within the object to the software (autoarranged should be checked), or to manually arrange the states (autoarranged should be unchcecked).

You can hide a state from the view by unchecking it in the States tab.
