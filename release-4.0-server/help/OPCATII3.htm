3. OPCAT II Main Window 

The main window of OPCAT II consists of 5 elements:
1. Menu bar - which includes Project, Edit, View, OPM Notation, Scaling, Comment, Generation, Format, Window, Help.
2. Standard toolbar - which includes new project, open project, close project, save project, print current OPD, cut a thing, copy a thing, paste a thing, undo, redo, delete.
3. Working area - which includes the OPDs the user works on.

4. Repository manager - which includes all the things in the current project grouped by OPDs.

5. Notation toolbar - which includes the OPM symbols grouped by things, relations, and links.
