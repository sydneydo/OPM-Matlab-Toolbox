Document Generation and Template Handling

OPCAT II has an option to generate a document, describing the system (analysis, design...).
The generated document is in html format. The content of the document depends on the choices of the user. A full document will include general information about the system, OPDs, OPL paragraphs and full description of things (objects and processes), links and relations.
The choices regarding the content may be saved as templates and used in different projects.
