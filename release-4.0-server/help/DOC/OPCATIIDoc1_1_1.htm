1.1 Choosing Contents

While choosing the contents of a document you have to decide what information is relevant. The document has 3 parts:

General Information: this part contains 12 fields, that describe the project. Select the ones you want to be shown in the document. If none of the fields are selected, this part will not appear in the document.

OPD & OPL: in this part you have to decide whether you want to have OPDs and/or OPL paragraphs in the document. You can have both, one or none of them. 

For OPDs you have the following options:

  • All levels - insert all the diagrams in the project (from root down) to the generated document.

  • Until level - insert all the diagrams from root down until the specified level.

  • By Name - insert the relevant diagrams, selected from a list.

  • None - insert no diagrams into the document.

For OPL paragraphs you have the following options:

  • All levels - insert all the OPL sentences of the project (one OPL paragraph with all the information, separated from the diagrams).

  • Until level - insert all the OPL paragraphs from root down until the specified level (separated from the diagrams).

  • By Name - insert the OPL paragraphs, selected from a list of diagram names described by the paragraph (separated from the diagrams).

  • None - insert no OPL paragraphs into the document.

  • According to OPD - under each selected diagram in the OPD part insert the relevant OPL paragraph.

The default in this part is All the OPD levels, with OPL paragraphs according to them.

Element Dictionary: in this part you should select what are the relevant fields for your document describing objects, processes, links and relations.

  • Things (objects and processes) - choose the relevant fields. If none of the fields are selected this part will not appear at all.

  • Links - choose the relevant link types and the relevant fields. If none of the types are selected this part will not appear at all.

  • Relations - choose the relevant relation types. If none of the types are selected this part will not appear at all.
