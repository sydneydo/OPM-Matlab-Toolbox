2. Templates Handling

In order to make the generation of a document easier OPCATII has an option to save and use templates for documents.
A Template is a set of choices regarding the contents of a document. It is general, and may be used in different projects. Select the "Handle Template" option in the Generation> Documents menu. The next window will be opened:

This window contains the following options:

2.1 Create a new Template

2.2 Edit a template

2.3 Delete a template
