6. Activity Diagrams

None -
The XMI file will not include any Activity Diagram.
From Top Level Down -
The user should enter an integer to define the level number. The XMI file will include Activity Diagrams for processes until the required level.
By Process -
The user should select at least one process among those that appear in the window. For the selected process group, one of the following options should be chosen:
All Levels Down -
The XMI file will include Activity Diagrams for the selected processes and all their sub-processes.
Levels down -
The XMI file will include Activity Diagrams for the selected processes and their sub-processes until the specified level (including).
