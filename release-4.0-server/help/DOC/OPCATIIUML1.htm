OPM to UML - Introduction

UML Generator enables generation of UML diagrams from OPM diagrams by creating XML file. The output XML file can be opened by any CASE tool that supports Unisys XMI, for example Rational Rose. The UML diagrams that can be created are:

1. Use Case diagram

2. Class diagram

3. Sequence diagrams

4. Deployment diagram

5. Statechart diagrams

6. Activity diagrams

It is possible to choose more than one of the diagrams. When pressing the "Generate" button the location window will appear.
