2.2 Edit a Template 

In order to insert changes into an existing template, choose the "Handle Template" option in the Generation>Documents menu, select the template you wish to edit from the list of existing templates and press "Edit".
You will see the same window like in making a new template. The name of the template will appear on top of the screen and the relevant fields will be selected.
Insert your changes and press the "Save" button.
