1. Use Case Diagram

It is possible to choose one of 4 options:

  • Root Level Only - the XMI file will include the elements of the root level only (SD - level).

  • All OPD Levels - the XMI file will include the elements in the entire system (OPD-set).

  • Until level - the user should enter an integer. the XMI file will include the elements which appear in all the diagrams from the top level (SD) until the specified level (including).

  • None - the XMI file will not include a Use Case Diagram.
