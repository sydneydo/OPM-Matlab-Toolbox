5. OPM Notation Operations 

    5.1 Add Object/Process

In order to add a new thing to the opened OPD, click its symbol (object or process) from the notation toolbar, or from the menu bar (OPM Notation > Thing > Object or Process). Then click on the appropriate location in the screen (where you want the thing to appear) and fill in the thing name (in the General tab). You can also specify its essence (physical or informatical), origin (environmental or systemic), scope (private, protected, or public) and an additional url. Url is used for linking additional files and documents to the Thing. The url could be invoked by pressing the Control key on the keyboard with the Left Mouse button, while the Thing is selected.

For objects, you can also specify its initial value and type. By default each object is a compound object (i.e. consists/characterizes of other things). If you want to specify a simple object, uncheck the compound object, and choose simple type (boolean, integer, char, etc.), advanced type (char[], time, date, etc.), or custome type (the objects declared in the current project).

In addition to the General tab, you can change the border, fill and text colors, and the text position in the Misc tab by clicking on the appropriate fields.

For objects, there are two additional tabs: Keys & Indices and States. In the Key & Indices tab you can specify if the current object is persistent (i.e. stored in the application database), a key, and/or an index. For indices specify the index name and order. You can also write the object description in this tab. You can control object states from the State tab, after confirming the object addition first, as described in 5.2 .

For processes, there are two other additional tabs: Description & Body and Activation Time. In the Description & Body tab you can write the process description and body (in C++, Java, etc.). In the Activation Time tab you can specify activate time constraints for the process (minimal execution time and maximal execution time) in terms of years, days, hours, etc.
