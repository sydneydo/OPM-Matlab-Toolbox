2. OPCAT II System Requirements and Installation

OPCAT II minimal requirements are:

1.      200 MHz processor.

2.      64 MB SDRAM.

3.      Java version 1.3 (or appropriate JRE).

OPCAT II installation file is an exe file. Double click it and specify the desired OPCAT II folder. The installation creates an icon on the desktop and a folder in the programs menu. For executing OPCAT II, double click opcat icon, or choose Start > Programs > Opcat > Opcat.
