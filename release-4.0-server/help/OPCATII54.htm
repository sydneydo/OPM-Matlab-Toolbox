5. OPM Notation Operations 

    5.4 Add Procedural Link

In order to connect two things with a procedural link, you should add the things first, and click the required link symbol. Then click the source thing and drag it towards the destination thing. Afterwards you can double click the link in order to specify its condition and path. In addition, you can specify reaction time constraints in the Reaction Time tab. In the Misc tab of the link properties you can specify the link border, fill, and text colors.

As for the unidirection and bidirectional relations, you can arrange links manually by first clicking the right button of the mouse, then unchecking the autoarrange property, and draging the relevant edge. The edge can be dragged to another object or process (if OPM rules allow it). In this case, the new object/process become the new source/destination for the link.
