3. Main Entities 

    3.1 Objects

An object is a thing that has a potential of stable, unconditional physical or mental existence. 

Objects are graphical represented by rectangulars. 

An informatical object is a piece of informatics, graphically denoted  by a rectangle.

A physical object  consists of matter. Therefore, in the broad sense, physical objects are tangible. Physical objects are graphically denoted by shaded rectangles.

An environmental object  affects the system, yet it is not a part of it. Environmental, informatical objects are denoted by dashed rectangles, while enironmental, physical objects are denoted by dashed, shaded rectangles.
 
 

Customer is an Environmental, physical object.

Account is a systemic, informatical object.
