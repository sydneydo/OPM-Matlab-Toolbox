5. Procedural Links 

    5.1 Enabling Links

Enabler  is an object  that is not changed through the process  occurence, yet essential to its execution.

Enabling link is a prosedural link that connects a process with an enabler of that process. There are three main types of enabling links: agent link, instrument link, and condition link.

An Agent is an intelligent enabler  (human), who controls process executions in the way that suites his/her needs. Agent links, denoted by black circle headed arrows, connect agents and the processes they enable. 

Customer handles Transaction Executing.

 

An Instrument is an enabler  that is not agent, e.g., tools, data, etc. Instrument  links, denoted by white circle headed arrows, connect instruments and the processes they enable. 

Transaction Executing requires Bank.

 

Condition links connect processes with object states. The meaning is of an if sentence: The process executes if and only if the object is in the certain state. Contrarily, the meaning of an instrument link is of "wait until". Condition links are symbolized by circle headed arrows with "c" literal inside.

Transaction Executing is guarded by Approval is obtained.
