3. Main Entities 

    3.3 Processes

A process is a pattern of transformation (consumption, change, or generation) that objects undergo.

Processes are the only thing that cause creation (construction) or elimination (destruction) of objects, as well as changes in objects' states. As long as no process  acts on the object, the object persist in its current state.

A process name should end with "ing", and if it doesn't the reserved word Process should be appended to it. An object name is forbidden to end with "ing", and if it does the reserved word Object should be appended to it. 

Processes are  graphically denoted by ellipses. The ellipse might be dashed to indicate environmental processes, and/or shaded to denote physical  processes.

Transaction Executing is a process.
