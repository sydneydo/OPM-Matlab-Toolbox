6. Scaling Mechanisms 

6.2 Unfolding/Folding

The Hierarchy Mode  of Scaling  operates on a thing by Folding  and Unfolding  it. Unfolding  reveals a set of lover-level things that are hierarchically below a relatively higher-level thing. The result of unfolding is a tree, the root of which is the thing being unfolded. 

Cash Card is unfolded to its features (attributes). Folding this diagram will show only the root, Cash Card.
