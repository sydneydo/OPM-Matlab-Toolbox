2.   Main Entities Recognition

 

2.1.        Object

o       When you draw a rectangle its better that the edges will intersect.

o       The gap between two segments should be small.

o       If a rectangle was drawn in a zoomed-In object you should choose an object in the popup toolbar.

 

2.2.        Process

o       When you draw an ellipse its better that the edges will intersect.

2.3.        State

o       When you draw a state (rectangle) in a zoomed-In object you should choose a state in the popup toolbar.

 

3.   Structural Relations Recognition

3.1.        †Relation

o       The triangle of the Relation can be drawn in several segments.

o       When you draw a Relation its better that the beginning and the end of the relation will intersect with the source and destination entities correspondingly.

 

3.2.        Bidirectional Relation

o       A Bidirectional Relation can be drawn in one, two or three segments.

 

4.   Procedural Links Recognition

4.1.        Link

o       The arrow head can be drawn in several segments.

o       When you draw a Link its better that the beginning and the end of the Link will intersect with the source and destination entities correspondingly.

 

Segment definition - The line that is drawn on the screen from the moment you press the mouse/pen till the moment you release it.
