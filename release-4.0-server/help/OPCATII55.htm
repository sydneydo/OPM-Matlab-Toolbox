5. OPM Notation Operations 

    5.5 Delete Object/Process

You can delete an object or a process only if it is not connected to any other thing (by relations or links), i.e. if you wish to delete an object or a process you first have to delete all of its relations and links, as described in 5.6. Click the required thing and then press the delete button of the keyboard. You can also open the mouse right button menu and click the delete option.
