Import - Overview

The import operation in Opcat is to allow several developers to work on a single model and incorporate their models into an overall global model. It allows an external model to be assimilated into a base model, copying all the properties and attributes of elements of the external model.

Aftr importing, the imported model is read/write

Note that import operation does not create a live link to the imported model. If the imported model is changed, the change will not be visible in the global model until it is imported again.

It is advised to hold a copy of the global model before importing, in case the imported models are changed and import has to be preformed again See Import best practices for more information
