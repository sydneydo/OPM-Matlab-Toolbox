The supported sentences

This is the list of the sentences, which are supported by the OPCAT2.

Relation sentences: 

        General Sentence:

 

        1.     (Cardinality) Object(s) relates to/tag (Cardinality) Object(s).

        2.     (Cardinality) Object(s) relates to/tag (Cardinality) Object(s) and (Cardinality) Object(s).

        3.     (Cardinality) Object(s) relates to/tag (Cardinality) Object(s), … , (Cardinality) Object(s), and (Cardinality) Object(s).

        4.     (Cardinality) Object(s) and (Cardinality) Object(s) are equivalent.

        5.     (Cardinality) Object(s) and (Cardinality)(s) Object tag.

Aggregation - Participation Sentence:

 

1.     (Cardinality) Object(s) consists of (Cardinality) Object(s).

2.     (Cardinality) Object(s) consists of (Cardinality) Object(s) and (Cardinality) Object(s).

3.    (Cardinality) Object(s) consists of (Cardinality) Object(s), … , (Cardinality)  Object(s),   

      and (Cardinality) Object(s).

 

Generalization - Specialization Sentence:

 

1.     Object is a/an Object.

2.     Process is Process.

 

Classification - Instantiation Sentence:

 

1.     Object is an instance of Object.

2.     Process is an instance of Process.

3.     Object and Object are instances of Object.

4.     Process and Process are instances of Process.

5.     Object, …, Object, and Object are instances of Object.

6.     Process, …, Process, and Process are instances of Process.
